fizzBuzz
Hello, welcome to the fizzBuzz game. In this game you will input a number into the search bar and then guess if it will return fizz, buzz, or fizzBuzz.

    Rules:

  • If a number is divisible by 3 then it will return fizz.
  • If a number is divisible by 5 then it will return buzz.
  • If a number is divisible by 15 then it returns fizzBuzz.
  • You will have 60 seconds you try and guess what a number will return.
  • Your final score will be added to the highscores list if it is in the top 5 highscores.
